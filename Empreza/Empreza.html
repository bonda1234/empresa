Que tipos de softwares distribuimos?

Un software que les permita comunicarse internamente a través de chat o videoconferencias.

Telegram

Es un programa con un diseño que parece clonado de WhatsApp, pero que incluye más opciones de seguridad y privacidad además de ser gratuito y estar programado con software libre. Permite realizar llamadas de VoIP, además de la opción de invitar a otros a mantener chats privados con altos niveles de seguridad en las comunicaciones.


Este software no necesita instalación de parte de nuestro pc , pero si en nuestro celular a través de la play store para poder vincularlo a través de un código QR en la página de telegram web. https://web.telegram.org/k/

Un software que les permita hacer presentaciones o videos para las reuniones

Google Slides

Presentaciones de Google es un programa de presentación incluido como parte de un paquete informático de software gratuito basado en la web que ofrece Google dentro de su servicio Google Drive.

Un software que les permita manipular audio e imagenes

Blender

Blender, uno de los mejores programas para editar vídeos gratis, está disponible para Windows, Mac y Linux. Es un programa para editar vídeos de código abierto que es completamente gratuito. Blender fue diseñado como una suite de animación en 3D, pero viene con un software de edición de vídeo muy útil. El editor de vídeo gratis de Blender debería ser suficiente para la mayoría de tus vídeos.



Un software que permita guardar y gestionar base de datos

MongoDB

MongoDB es un sistema de base de datos NoSQL orientado a documentos de código abierto y escrito en C++, que en lugar de guardar los datos en tablas lo hace en estructuras de datos BSON (similar a JSON) con un esquema dinámico.
